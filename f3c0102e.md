## What this adds to lyca

Three capabilities, all in `lyca.py`:

| Symbol | Kind | Purpose |
|---|---|---|
| `MODEL_REGISTRY` | module-level `list[dict]` | Curated metadata for every `litert-community` LLM |
| `models(filter?)` | function | List / filter the registry in a readable table |
| `syscheck()` | function | Detect RAM, CPU, platform, GPU availability |
| `recommend()` | function | Cross-join registry with syscheck, rank fits |
| `download(repo, file?, dest?)` | function | HF download with resume + progress |
| `Chat.from_hf(repo, file?)` | classmethod | `download` + `Chat(...)` in one call |

New hard dependency: **`huggingface_hub`** (adds to the four existing ones).

## `syscheck()` — system detection

Returns a plain `dict`. Uses only stdlib + optional `psutil` for RAM on all platforms.
No third-party system-info libraries.

```python
{
  "platform":   "macOS",          # "macOS" | "Linux" | "Windows"
  "arch":       "arm64",          # "arm64" (Apple Silicon) | "x86_64" | "aarch64"
  "cpu":        "Apple M4 Max",   # brand string or "unknown"
  "ram_gb":     64.0,             # physical RAM in GB (float, 1 decimal)
  "free_ram_gb": 42.1,            # available RAM right now
  "gpu":        "metal",          # "metal" (macOS arm64) | "cuda" | "none"
  "gpu_vram_gb": None,            # None for Metal (unified), float for CUDA
  "apple_chip": "M4",             # "M1"|"M2"|"M3"|"M4"|None
  "apple_tier": 3,                # 1=base, 2=Pro, 3=Max, 4=Ultra — None if not Apple Silicon
}
```

### Platform detection logic

```
macOS:   subprocess("sysctl -n hw.memsize")  → total RAM bytes
         subprocess("sysctl -n machdep.cpu.brand_string") → CPU brand
         parse "Apple M4 Max" → chip="M4", tier=3 (Max)
         GPU: "metal" if arch=="arm64" else "none"
         gpu_vram_gb: None (unified memory — not separately addressable)

Linux:   /proc/meminfo → MemTotal + MemAvailable
         /proc/cpuinfo → model name
         check nvidia-smi → CUDA + VRAM
         or: psutil.virtual_memory() as universal fallback

Windows: subprocess("wmic ComputerSystem get TotalPhysicalMemory")
         or: psutil.virtual_memory()

Fallback (always works): psutil.virtual_memory() if psutil is importable
```

### Apple tier parsing

```python
import re

def _parse_apple_chip(brand: str) -> tuple[str|None, int|None]:
    "Return (chip_name, tier) from a CPU brand string like 'Apple M4 Max'."
    m = re.search(r"Apple (M\d+)\s*(Pro|Max|Ultra)?", brand)
    if not m: return None, None
    chip = m.group(1)                             # "M4"
    tier_map = {None: 1, "Pro": 2, "Max": 3, "Ultra": 4}
    return chip, tier_map[m.group(2)]
```

### `gpu_usable(spec: dict) -> bool`

Private helper: returns True if GPU backend can be used.
- macOS arm64 → Metal always available
- Linux with CUDA → True if nvidia-smi returns without error
- Otherwise → False

## `recommend()` — model ranking

```python
def recommend(
    min_tps: float = 5.0,     # minimum acceptable decode speed (tokens/sec)
    task: str = "chat",        # filter by task type
    verbose: bool = True,      # print table if True
) -> list[dict]:               # sorted models that fit this system
```

### Fit logic (applied in order)

1. `fits_ram`: `spec["free_ram_gb"] >= entry["min_ram_gb"] * 1.15` (15% headroom)
2. `fits_disk`: `entry["size_gb"] is not None` (skip unknown sizes for now)
3. `fits_task`: `entry["task"] == task`
4. `expected_tps`: if `gpu_usable` and `entry["mac_gpu_tps"]` → use GPU tps; else CPU tps
5. `meets_speed`: `expected_tps is None or expected_tps >= min_tps` (None = unknown → include)

### Ranking key

```python
# Higher is better
score = (expected_tps or 20) * (1 + 0.1 * len(entry["tags"]))
```

## `download()` — HF download with resume

```python
def download(
    model_id: str,              # Registry id ("gemma4-e4b") OR full HF repo ("litert-community/...")
    filename:  str = None,      # Specific .litertlm file; inferred from registry if None
    dest:      str = None,      # Local directory; defaults to ~/.cache/lyca/
    token:     str = None,      # HF token for gated models
    force:     bool = False,    # Re-download even if file exists
) -> str:                       # Absolute path to downloaded .litertlm file
```

### Implementation details

```python
from huggingface_hub import hf_hub_download

# 1. Resolve repo + file from registry or direct string
# 2. Determine dest: Path(dest or Path.home()/".cache"/"lyca") / repo_name
# 3. Call:
path = hf_hub_download(
    repo_id        = repo,
    filename       = file,
    local_dir      = dest,
    resume_download= not force,
    token          = token,
)
# 4. Return absolute path string
```

**Auto-discover filename** when `file is None and entry["file"] is None`:
```python
from huggingface_hub import HfApi
files = [f.rfilename for f in HfApi().model_info(repo, files_metadata=False).siblings
         if f.rfilename.endswith(".litertlm")]
# pick the smallest (most general) one if multiple
```

**Progress**: `hf_hub_download` shows a tqdm bar by default. No extra work needed.

**Already downloaded**: if the target path exists and `force=False`, skip download and
print `"✓ Already downloaded: {path}"`.

## `Chat.from_hf()` — classmethod

```python
@classmethod
def from_hf(
    cls,
    model_id:  str,         # Registry id OR full HF repo
    filename:  str = None,
    dest:      str = None,
    token:     str = None,
    **kwargs               # forwarded to Chat.__init__ (sp, tools, hist, ...)
) -> "Chat":
    "Download model from HuggingFace then return a ready Chat."
    path = download(model_id, filename=filename, dest=dest, token=token)
    return cls(path, **kwargs)
```

Same classmethod on `AsyncChat` via `@delegates`.

## `register_model()` — runtime registry extension

```python
def register_model(entry: dict) -> None:
    """Add a model entry to MODEL_REGISTRY at runtime.

    Required keys: id, repo, file, size_gb, min_ram_gb, task.
    Optional keys: family, params, quant, gpu_ram_gb, mac_cpu_tps, mac_gpu_tps, tags.
    """
    required = {"id", "repo", "file", "size_gb", "min_ram_gb", "task"}
    assert required <= set(entry), f"Missing keys: {required - set(entry)}"
    MODEL_REGISTRY.append(entry)
```

## Style Rules (same as SPEC.md — enforce)

1. `MODEL_REGISTRY` is a plain `list[dict]` — **no dataclass, no TypedDict**
2. `syscheck()` returns a plain `dict` — not a dataclass
3. Table output uses **stdlib only** (`str.ljust`, `str.rjust`, `print`) — no `rich`, no `tabulate`
4. `download()` is a **pure function** (no class state) — importable standalone
5. `hf_hub_download` is the only download primitive — **do not reimplement progress bars**
6. All subprocess calls use `subprocess.run(..., capture_output=True, text=True, timeout=5)` — never `os.system`
7. GPU detection must not crash on systems without `nvidia-smi` — wrap in `try/except`
8. `from_hf` is a classmethod, not a free function, so `Chat.from_hf(...)` is the call site
